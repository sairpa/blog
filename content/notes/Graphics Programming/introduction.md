### Handling Attachments

I've created an `attachments/` folder right next to this file (`content/notes/Graphics Programming/attachments/`). You can drop all your images, diagrams, or PDFs in there to keep your workspace clean.

To display an image from that folder inside your markdown, use a relative link like this:

```markdown
![My Diagram](attachments/my-diagram.png)
```
